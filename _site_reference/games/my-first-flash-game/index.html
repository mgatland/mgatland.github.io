My First Flash Game

October 17, 2009

Click on the skull to start, then use the arrow keys to move.

My first flash game, built following the great ‘Avoider Game’ tutorial found at Michael J Williams’s website.

I had to adapt the tutorial slightly because I was not using the Flash editor. I worked solely in FlashDevelop (using the Flex SDK to compile).
